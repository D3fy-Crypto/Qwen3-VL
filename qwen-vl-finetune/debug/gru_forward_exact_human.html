End-to-End Flow

1) Annotation DB

Read trajectory actions from annotations.json grouped by video_id step.

2) Frame Slots

Select 8 frames and compute per-slot frame cutoffs.

3) GRU Prefix Build

Build prefix action sequence per slot and convert to 7D motion features.

4) GRU + Projector

Encode with GRU, pick valid slot output, project to 4096D motion token embeddings.

5) LLM Fusion

Insert projected motion tokens at <motion> positions in prompt embeddings.
